 Automatic Water Level Controller using Verilog

 📌 Overview
This project implements an automatic water level controller using Verilog HDL. It uses a Finite State Machine (FSM) to monitor water levels and control a pump accordingly.

 🎯 Objectives
- Automate water level monitoring  
- Control pump based on tank levels  
- Prevent overflow and dry running  
- Implement FSM-based control logic  

🧠 Working Principle
The system uses three sensors:
- Low level sensor  
- Mid level sensor  
- High level sensor  

Operation:
- Low/Mid level → Pump ON  
- High level → Pump OFF  

 ⚙️ Methodology
- FSM-based design  
- Clock-driven state transitions  
- Sensor-based input monitoring  
- Testbench for simulation  

 🛠️ Components (Simulation)
- Verilog HDL  
- Testbench  
- Simulation tool (ModelSim / EDA Playground)

🔄 Features
- Automatic pump control 💧  
- Prevents overflow  
- Efficient water usage  
- Simple and reliable design  

 🌍 Applications
- Water tanks  
- Irrigation systems  
- Industrial water management  
- Home automation  

 ✅ Advantages
- Saves water  
- Reduces manual effort  
- Prevents tank overflow  
- Energy efficient  

 ⚠️ Limitations
- Sensor dependency  
- Simulation-based system  
- No real hardware implementation  

 🚀 Future Scope
- IoT-based monitoring  
- Mobile app control  
- Smart irrigation systems  


📌 Conclusion
This project demonstrates an efficient FSM-based water level controller that automatically manages pump operation based on sensor inputs.
